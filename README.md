# MCP Secure DB Agents

Este repositório contém a documentação, implementação experimental e materiais de avaliação de uma arquitetura baseada em Model Context Protocol para permitir que agentes LLM realizem consultas seguras a bancos de dados relacionais.

O projeto investiga o uso do MCP como camada intermediária entre agentes LLM e bancos de dados, com foco em validação de SQL, controle de acesso, auditoria, mitigação de prompt injection, indirect prompt injection e tool poisoning.
